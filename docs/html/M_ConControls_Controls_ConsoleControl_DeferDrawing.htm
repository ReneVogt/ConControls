Type: IDisposable
An IDisposable that on calling Dispose releases the drawing barriere of this window.

Implements

IControlContainerDeferDrawing
Remarks
Use this to surround a block of code that changes multiple properties of this IControlContainer or its children.

using(DeferDrawing){....}

At the end of this block, drawing will be resumed again.
