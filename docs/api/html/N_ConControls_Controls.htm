Remarks
This namespace contains the central base classes for creating ConControls user interfaces.
These include the main ConsoleWindow (implementing the IConsoleWindow interface), the ConsoleControl base class for console UI controls and the IControlContainer interface.
It also provides the ControlCollection that can keep the child controls of a IControlContainer.
