IControlContainerDeferDrawing Method

Defers any drawing operation until the return value has been disposed of again.

Namespace:  ConControls.Controls
Assembly:  ConControls (in ConControls.dll) Version: 0.1.1
Syntax
C#
Copy
IDisposable DeferDrawing()

Return Value

Type: IDisposable
An IDisposable that on calling Dispose releases the drawing barriere of this window.
Remarks
Use this to surround a block of code that changes multiple properties of this IControlContainer or its children. This avoids redrawing the control for each changed property, and instead redraws the whole container when the returned value is disposed.
C#
Copy
ConsoleControl control = ...;
using(control.DeferDrawing())
{
    control.ForegroundColor = ConsoleColor.Green;
    control.BackgroundColor = ConsoleColor.Black;
}
At the end of this block, drawing will be resumed again.
