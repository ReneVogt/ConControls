Append
Appends the given text to the current text. If the caret is positioned at the end of the current text, it will be placed at the end of the appended text and ScrollToCaret will be called after to make sure the appended text is shown.

ScrollToCaret
Scrolls the content so that the caret (cursor) becomes visible. If the caret already is inside the displayed text, the scrolling will not be changed. If the caret is outside the displayed text, scrolling is adjusted so that the caret just appears in the client area.
