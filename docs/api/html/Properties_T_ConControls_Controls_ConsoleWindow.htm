ConsoleWindow Properties

The ConsoleWindow type exposes the following members.

Properties
  NameDescription
ActiveScreen
Gets or sets wether this window is the active console screen buffer. If set to true this window is the active screen. If set to false the original console screen buffer will be activated.
Area
A Rectangle representing the area (in character columns and rows) this IControlContainer occupies in the console screen buffer.
CloseWindowKey
The key combination to close the window.
Controls
The collection of child controls in this IControlContainer.
DefaultBackgroundColor
Gets or sets the default background color.
DefaultBorderColor
Gets or sets the default border color.
DefaultCursorSize
Gets or sets the default cursor size to use (0-100) for newly created focusable controls.
DefaultForegroundColor
Gets or sets the default foreground color.
DrawingInhibited
Determines if drawing is currently allowed or inhibited via DeferDrawing or the Visible property.
Enabled
Gets wether this control or IControlContainer is enabled.
ExitCode
The exit code the window was closed with.
FocusedControl
Gets or sets the currently focused control on this window.
FrameCharSets
The FrameCharSets provider to use to draw frames.
IsDisposed
The window has been disposed of.
Location
The location of this IControlContainer in character columns and rows from the upper left corner of the console screen buffer.
Size
The size of this IControlContainer in character columns and rows.
SwitchConsoleBuffersKey
The key combination to switch between the window and the original console screen buffer.
SynchronizationLock
A synchronization object to synchronize conosle operations.
Title
The title of the console window.
Visible
Gets wether this control or IControlContainer is visible.
Window
The main IConsoleWindow this IControlContainer belongs to.
