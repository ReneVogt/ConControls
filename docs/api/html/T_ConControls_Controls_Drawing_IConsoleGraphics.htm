Remarks
This interface represents an abstraction layer for ConsoleControl instances to draw on the console screen buffer. It is implemented by the internal class ConsoleGraphics.
An instance of this interface must be provided by IConsoleWindow.GetGraphics to enable controls to draw themselves.
