ConsoleControl Properties

The ConsoleControl type exposes the following members.

Properties
  NameDescription
Area
The effective total area of the control. This is the area the control effectivly fills in the console screen buffer after applying layout and including borders.
BackgroundColor
Gets or sets the ConsoleColor to use for the background of this control.
BorderColor
Gets or sets the ConsoleColor to use for the border of this control.
BorderStyle
Gets or sets the BorderStyle of this control.
CanFocus
Determines wether this control can be focused or not.
Controls
The collection of ConsoleControls contained by this control.
CursorPosition
Gets or sets the cursor position for this control in character coordinates of the console screen buffer.
CursorSize
Gets or sets the cursor size for this control.
CursorVisible
Gets or sets wether the cursor is visible on this control. screen buffer.
DisabledBackgroundColor
Gets or sets the ConsoleColor to use for the background of this control when it is disabled. If this is null, the BackgroundColor value will be used.
DisabledBorderColor
Gets or sets the ConsoleColor to use for the border of this control when it is disabled. If this is null, the BorderColor value will be used.
DisabledBorderStyle
Gets or sets the BorderStyle of this control when it is disabled. If this is null, the BorderStyle value will be used.
DisabledForegroundColor
Gets or sets the ConsoleColor to use for foreground drawings when the control is disabled. If this is null, the ForegroundColor value will be used.
DrawingInhibited
Determines if the control can currently be redrawn, depending on calls to DeferDrawing to this control or its parents.
EffectiveBackgroundColor
Gets the current background color based on the state of Enabled and Focused properties.
EffectiveBorderColor
Gets the current border color based on the state of Enabled and Focused properties.
EffectiveBorderStyle
Gets the current border style based on the state of Enabled and Focused properties.
EffectiveForegroundColor
Gets the current foreground color based on the state of Enabled and Focused properties.
Enabled
Gets or sets wether the control is enabled or not.
Focused
Gets or sets wether this control is focused or not. Trying to set this to true though CanFocus returns false throws an InvalidOperationException.
FocusedBackgroundColor
Gets or sets the ConsoleColor to use for the background of this control when it is focused. If this is null, the BackgroundColor value will be used.
FocusedBorderColor
Gets or sets the ConsoleColor to use for the border of this control when it is focused. If this is null, the BorderColor value will be used.
FocusedBorderStyle
Gets or sets the BorderStyle of this control when it is focused. If this is null, the BorderStyle value will be used.
FocusedForegroundColor
Gets or sets the ConsoleColor to use for foreground drawings when the control is focused. If this is null, the ForegroundColor value will be used.
ForegroundColor
Gets or sets the ConsoleColor to use for foreground drawings.
Location
The location of this IControlContainer in character columns and rows from the upper left corner of the console screen buffer.
Name
Gets or sets the name of this control. If this is set to null or an empty or whitespace-only string, the name will be set to the name of the control's type.
Parent
The parent ConsoleControl that contains this control. The parent must be contained by the same IConsoleWindow. If this propery is null, the control is not displayed.
Size
The size of this IControlContainer in character columns and rows.
TabOrder
Gets or sets a number indicating the control's position in the Tab order.
TabStop
Gets or sets wether this control will be focused when using the Tab key to step through the window.
Tag
Gets or sets a tag object for this control.
Visible
Gets or sets wether the control should be visible (drawn) or not.
Window
The IConsoleWindow that contains this control.
