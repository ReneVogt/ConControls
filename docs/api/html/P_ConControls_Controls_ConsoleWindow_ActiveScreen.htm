ConsoleWindowActiveScreen Property

Gets or sets wether this window is the active console screen buffer. If set to true this window is the active screen. If set to false the original console screen buffer will be activated.

Namespace:  ConControls.Controls
Assembly:  ConControls (in ConControls.dll) Version: 0.2.1

Remarks
There seems to be no Windows API method to get the currently active console screen buffer. So this property does not really determine if the window's buffer is currently active. It rather returns the internal flag indicating wether the window instance tried to set its buffer as active console buffer or if it switched back to the original buffer.

If someone changed the active console screen buffer since the last set of ActiveScreen, the return value of this property may be misleading.
